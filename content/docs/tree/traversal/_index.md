## 一般树

不妨假设每个结点的值是一个字符串

```
              A
            / | \
           B  C  D
          /|     |
         E F     G
        /|\
       H I J
```

```go
type TreeNode struct {
	Children []*TreeNode
	Val string
}
```

也可以用数组或哈希表来代表结点，如[Trie前缀树的一个实现](/docs/design/implement-trie-prefix-tree)及[树的路径和问题探讨](/docs/tree/path-sum/d.go)中getPath函数的入参relations
